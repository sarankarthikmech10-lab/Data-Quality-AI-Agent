## ✨ Features

### 📊 Comprehensive Profiling
- **Statistical Analysis**: Instant computation of mean, median, standard deviation, skewness, and more.
- **Visual Dashboards**: Interactive charts for missing value distribution and column type frequency.
- **Deep Scanning**: Automatic detection of column types (Numeric, Categorical, Boolean, Datetime).

### 🤖 Intelligent Issue Detection
- **Outlier Detection**: Uses IQR and Z-score methods to flag statistical anomalies.
- **ML Anomaly Scoring**: Advanced reconstruction-error logic to find non-obvious data patterns.
- **Format Consistency**: Detects mixed casing, leading/trailing whitespace, and inconsistent date formats.
- **Severity Ranking**: Issues are categorized by severity (**High**, **Medium**, **Low**) to prioritize your workflow.

### 🛠 Automated Remediation
- **One-Click "Auto-Fix"**: Instantly apply all recommended strategies.
- **Smart Imputation**: Fills missing values using median/mode based on data type.
- **Deduplication**: Intelligent row-level deduplication.
- **Normalization**: Standardizes casing and trims whitespace automatically.

### 📈 Quality Scoring
- **Multi-Dimension Scoring**: Scores your data based on **Completeness**, **Uniqueness**, **Validity**, and **Consistency**.
- **Before/After Analysis**: View a side-by-side comparison of metrics before and after cleaning.

## 🛠 Built With

- **HTML5/CSS3**: Modern dark-mode glassmorphism UI.
- **Vanilla JavaScript**: High-performance data processing entirely in the browser.
- **Chart.js**: Dynamic data visualizations.
- **PapaParse**: Reliable CSV parsing.
- **SheetJS**: Multi-format support including XLSX and JSON.

---

## 📖 How to Use

1. **Upload**: Drag and drop your CSV, JSON, or XLSX file into the agent.
2. **Profile**: Review the initial data health and distribution charts.
3. **Analyze**: Let the agent identify quality issues and their root causes.
4. **Clean**: Choose "Auto-Fix" or manually approve each remediation step.
5. **Download**: Obtain your perfected, clean dataset instantly.
